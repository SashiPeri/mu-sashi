# Mu Sashi Design System

## Philosophy

The app should feel like entering a sacred dojo.

Not gamified.
Not corporate SaaS.
Not productivity software.

It should feel quiet, disciplined, premium, and timeless.

Primary emotions:
- Focus
- Mastery
- Progress
- Calm confidence

## Visual Language

- Black background (#000000)
- Dark zinc surfaces
- Deep crimson accents
- Very subtle borders
- Large whitespace
- Minimal shadows
- Typography carries hierarchy

## Components

### Cards
- Rounded 24px
- Thin borders
- Slight hover lift (2-3px)
- Gentle border glow on hover

### Buttons
- Minimal
- Text first
- Deep red only for primary actions

### Progress
- Thin progress bars (2-3px)
- No gradients
- Elegant animations

## Layout

Desktop:
- Left fixed sidebar
- Right content panel
- Maximum width around 1400px

Spacing:
- 8px system
- Generous padding
- Air between sections
