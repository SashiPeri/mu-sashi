# Mu-Sashi Design Reference

The images in this folder are the canonical visual language of the application.

Always review these images before making UI changes.

Rules:
- Preserve spacing and typography.
- Preserve the quiet, premium atmosphere.
- Do not introduce glassmorphism, neon gradients, or excessive animations.
- The +1 Rep interaction is the emotional center of the product.
- If generated ideas conflict with these screenshots, the screenshots take priority.

These images represent the intended design system, not merely inspiration.
